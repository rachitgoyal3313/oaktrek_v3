Accessibility Statement

OakTrek is committed to ensuring digital accessibility for people with disabilities. We are continually improving the user experience for everyone and applying the relevant accessibility standards.

Conformance Status

We aim to conform to the Web Content Accessibility Guidelines (WCAG) 2.1 Level AA. These guidelines explain how to make web content more accessible for people with diverse abilities.

Accessibility Features Implemented

  • Keyboard Navigation: The site can be navigated using a keyboard.
  • Screen Reader Compatibility: We strive to ensure compatibility with popular screen readers.
  • Alternative Text: Images have descriptive alt text.
  • Semantic Structure: Content is structured using appropriate headings and landmarks.
  • Color Contrast: Text and background colors generally meet contrast requirements.
  • Resizable Text: Text can be resized using browser controls without loss of content or functionality.

Ongoing Efforts

Accessibility is an ongoing effort. We regularly review our site and work with accessibility experts to identify and fix potential barriers.
